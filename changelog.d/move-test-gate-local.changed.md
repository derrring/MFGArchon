- **The authoritative test gate moved from GitHub to local.** The full suite runs in ~141 s locally but exceeded a 25-minute step budget on a GitHub runner; measured, coverage accounts for 1.5x of that and the runner itself for the rest. Online execution of the full suite was buying latency, not signal. `ci.yml` now runs a smoke subset (`test_core` + `test_config`, ~2 min) on PRs alongside the existing lint/format/ratchet/import checks; `nightly.yml` keeps the full run incl. `@slow`; releases keep the full run with coverage. New `scripts/local_ci.sh` is the pre-push gate -- ruff format, ruff lint over `mfgarchon/` **and** `tests/` (wider than CI's `--select F` on `mfgarchon/` alone), the fail-fast ratchet, and the full suite. **A GitHub-green PR has not had its tests run**; `local_ci.sh` must be run before every push.
