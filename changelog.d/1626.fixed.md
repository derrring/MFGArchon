- **Tutorial 02's congestion Hamiltonian is no longer physically inverted** (Issue #1626). The tutorial used `H = (1/2 + lambda*m)|p|^2`, putting the density in the numerator, so `|alpha*| = (1 + 2*lambda*m)|p|` *grew* with density -- crowds made agents move faster, and the notebook printed "Evacuation slowdown: -291.1%" directly contradicting its own narrative. Replaced with the Legendre transform of the congested Lagrangian, `H = |p|^2 / (2(1 + lambda*m))`, matching the convention the library's own `CongestionHamiltonian` already uses. The run now converges in 22 iterations (was 100, unconverged) and reports a 59.6% slowdown. Formulas and narrative updated in lockstep in both the notebook and the paired script; the script also picked up the shape-robustness pattern from #1625 that only the notebook had received.
